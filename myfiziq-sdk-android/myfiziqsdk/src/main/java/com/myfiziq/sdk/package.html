Contains the primary classes for the MyFiziq SDK.

MyFiziqSdkManager, MyFiziq, and MyFiziqActivity are the main entry points in to the SDK.

Steps required to use the SDK:
* Call MyFiziqSdkManager.getAppConf with the required token and environment parameters.
* Call sdkSetup with the API key and Secret Key.
* Start the MyFiziqActivity activity to begin a capture.
